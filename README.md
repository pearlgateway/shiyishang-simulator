# 世一上 Bin 模拟器（ShiYiShang Simulator）

回合制 LOL 上单讽刺网页小游戏。你扮演「世一上」Bin：单带吃线、激进压线、拒绝买眼，以及最核心的抉择——**队友打团，你 T 还是不 T？**

> 基地输赢不重要，嘴硬才是总冠军。

## 双模式

### MSI 2026 决赛（vs Zeus·蒙多，10回合 BO5）
- 专属机制「心之钢计数器」：你每拒绝一次TP、每被抓一次，Zeus的蒙多就叠钢
- 决胜局团战胜率随心之钢衰减——700层以上三段Q全中血条纹丝不动
- 结局：让二追三 / 奎桑提不TP / 捧杯（陌生）

### EWC 八强赛（vs DK·Siwoo，6回合 BO3）
- 专属机制「被单杀计数 0/4」：激进换血多半被「高配版你」反手教育
- 满4次被单杀立即触发「米勒破防」结局
- 结局：米勒破防 / 高配Bin / 险胜但评分4.0 / LPL全军覆没

## 数据严谨性设计

- 团战悬挂时做任何其他操作 = 事实拒团，正常结算龙/经济/心态
- 决胜团最后一回合100%必刷；一方4龙后改刷大龙团（计经济差）
- 参团率分母为0时显示「—」；团队经济差全程可见；结算含真实分均经济

## 技术栈

React 19 + TypeScript + Vite 7 + Tailwind CSS 3

## 本地运行

```bash
npm install
npm run dev
```

构建：

```bash
npm run build  # 产物在 dist/
```
